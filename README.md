## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/swapnilbharate/Swapnil-Portfolio.git
```

Navigate to the project folder:

```bash
cd Swapnil-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```
